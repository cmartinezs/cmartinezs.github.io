## Proyecto destacado

### KeyGo: plataforma de autenticación como servicio

**KeyGo** es un sistema SaaS diseñado para gestionar autenticación de usuarios y aplicaciones de forma simple, segura y escalable.

Características principales:

- Multi-tenant, con organizaciones independientes.
- Gestión de usuarios y roles.
- Autenticación centralizada.
- Tokens para aplicaciones y usuarios.
- Arquitectura escalable pensada para startups.

**Estado:** próximamente disponible.
